# File Extension Block

파일 업로드 시 확장자 기반으로 차단하는 웹 서비스입니다.  
고정 확장자(체크박스)와 커스텀 확장자(추가/삭제)를 관리하고, 업로드 테스트로 차단 여부를 확인할 수 있습니다.

## 주요 기능
- 고정 확장자 관리
    - 고정 확장자 목록 조회
    - 고정 확장자 차단 여부(체크) 업데이트
- 커스텀 확장자 관리
    - 커스텀 확장자 추가 (최대 200개, 길이 20자 제한)
    - 커스텀 확장자 삭제
    - 커스텀 확장자 목록 조회
- 업로드 테스트
    - 파일 업로드 시 확장자를 추출해 차단 여부 판단 (대소문자 무시)
